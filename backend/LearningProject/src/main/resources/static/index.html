User Management API

API endpoints for managing users in the learning platform

User Endpoints

Register Single User

POST /api/v1/user/register

Request Body

Field Type Description
email String Must be unique
username String Must be unique
firstName String User's first name
lastName String User's last name
password String Will be salted

Example Request

{
    "email": "student@university.edu",
    "username": "jdoe2023",
    "firstName": "John",
    "lastName": "Doe",
    "password": "securePass123"
}

Responses

Code Description
200 User registered successfully
409 Email or username already exists
400 Invalid request data

Batch User Registration

POST /api/v1/user/registerMultipleUsers

Request Body

Array of user objects (same structure as single registration)

Responses

Code Description
200 All users registered successfully
207 Partial success (check response for details)
409 All registrations failed

Example Response (207)

{
    "successes": [
        "User student1@university.edu registered"
    ],
    "errors": [
        "User student2@university.edu: email exists",
        "User jsmith: username exists"
    ]
}

Get User

GET /api/v1/user/{id}

Responses

Code Description
200 Returns user data
404 User not found

Example Response

{
    "id": 123,
    "email": "student@university.edu",
    "username": "jdoe2023",
    "firstName": "John",
    "lastName": "Doe",
    "createdAt": "2023-11-15T10:30:00Z"
}
